## 🚀 프로젝트 실행 방법

프로젝트를 클론하거나 다운로드합니다.

로컬 서버 또는 브라우저에서 index.html 파일을 실행합니다.

이미지나 CSS 파일이 정상적으로 불러오지 않는 경우, 경로에 포함된 ../를 제거한 후 다시 확인해 주세요.
